# Vinted Notifications Unlocker

Diese Browser-Erweiterung für Chrome und Microsoft Edge erweitert die Benachrichtigungsseite von Vinted um ein Seiten-System und einstellbare Item-Anzahl pro Seite.

## Features
- Blätterfunktion für Benachrichtigungen (Seiten)
- Einstellbare Anzahl an Benachrichtigungen pro Seite
- Einstellungen über das Optionsmenü

## Installation
1. Lade den Ordner `extension` als ZIP herunter oder klone das Repository.
2. Öffne `chrome://extensions` (Chrome) oder `edge://extensions` (Edge).
3. Aktiviere den Entwicklermodus.
4. Klicke auf „Entpackte Erweiterung laden“ und wähle den `extension`-Ordner aus.

## Veröffentlichung
- Die Erweiterung ist kompatibel mit dem Chrome Web Store und Microsoft Edge Add-ons Store.
- Icons (16x16, 48x48, 128x128) müssen im Ordner `icons/` ergänzt werden.

## Hinweise
- Die Erweiterung funktioniert nur auf der Seite [https://www.vinted.de/member/notifications](https://www.vinted.de/member/notifications).
- Für weitere Anpassungen siehe `content.js` und `options.js`.
